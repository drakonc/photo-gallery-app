# Api: Subir Fotos

## photo-gallery-app

> Esta es una aplicacion BackEnd con la cual podras subir Fotos a tu FrontEnd

## Instalación

```bash
$ npm install
```

## Ejecutar la app

```bash
# producción
$ npm run start

# development
$ npx nodemon

# watch mode
$ npm run start:dev 
```
## Borra Carpeta Distribucion 

```bash
# Clean dist
$ npm run clean
```

## Pasar a Producción

```bash
$ npm run build
```
